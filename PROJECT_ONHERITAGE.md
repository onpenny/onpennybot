# OnHeritage 項目規劃

## 項目概述

**OnHeritage** - 一站式遺產管理平台，解決資產信息孤島、繼承主體不明、繼承流程繁瑣等核心痛點。

**技術棧**：
- ⚛️ Next.js (React) + TypeScript
- 🎨 Tailwind CSS
- 📦 GitHub: https://github.com/onpenny/onpennybot

---

## 一、項目價值主張

### 解決的核心痛點
1. **資產信息孤島化** - 銀行、保險、券商等機構資產無統一歸集入口
2. **繼承主體不明確** - 無標準化家族譜系存儲，繼承權確權難
3. **生前資產遺忘漏記** - 多類投資、另類資產零散分布
4. **繼承流程非標繁瑣** - 不同地區/法域法律差異大
5. **遺產捐獻無閉環** - 捐贈意願落地難、透明度低
6. **跨境繼承壁壘高** - 海外資產繼承困難
7. **隱私與存儲矛盾** - 安全性與可繼承性難兼顧

### 核心基礎功能
1. 全品類資產管理（API同步 + 手動錄入）
2. 智能家族譜系追蹤（可視化族譜）
3. 自動觸發與繼承通知（生命狀態核驗）
4. 遺囑與分配規則管理（法域合規模板）
5. 遺產捐獻全流程管控（意願登記 → 執行追溯）
6. 一站式繼承工具（材料清單、流程指引）
7. 隱私安全體系（端到端加密、多重驗證）

## 二、商業模式

### （一）訂閱制收費
| 版本 | 功能 | 定價 |
|------|------|------|
| 免費版 | 基礎族譜、限量手動錄入、基礎雲存證 | $0 |
| 基礎版 | 全量手動錄入、標準遺囑模板、基礎繼承通知 | 年付 |
| 進階版 | 金融機構 API 自動同步、多繼承人、條件觸發、區塊鏈存證 | 年付 |
| 尊享版 | 跨境資產、專屬顧問、定製分配規則、私密雲櫃 | 年費/終身 |
| 家族套裝 | 多成員賬號捆綁，統一管理 | 優惠價 |

### （二）增值服務收費
- 法律與公證服務（律師、公證處合作，分成）
- 遺產清算與代執行（按資產規模比例收費）
- 非標資產評估（收藏品、藝術品、股權、IP評估）
- 稅務籌劃服務（遺產稅、贈與稅優化）
- 沉睡資產尋回（成功追回後按比例分成）

### （三）B端合作授權
- 金融機構白標系統模塊（SaaS年費）
- 政企項目制（民政部門、公證處、基金會）
- 跨境服務對接（海外律所、財富機構）

### （四）衍生變現
- 數字遺產與文創（族譜畫冊、家族傳承檔案）
- 行業數據服務（合規匿名化報告）
- 精準產品導流（終身壽險、家族信託）

---

## 三、競爭優勢

1. **譜系 + 資產雙綁定** - 關係與分配規則深度關聯
2. **主動觸發機制** - 合規數據核驗實現主動通知
3. **全法域適配** - 覆蓋境內多區 + 主流海外國家
4. **捐贈全閉環** - 從意願登記到執行追溯一體化
5. **生生死後全週期** - 規劃、預案、執行、傳承

---

## 四、合規與風險把控

- 遵循各地數據安全法規（PIPL、GDPR）
- 明確電子遺囑、區塊鏈存證法律效力邊界
- 僅與持牌法律、金融、評估機構合作
- 完善的授權、註銷、數據刪除機制
- 流程全留痕，降低法律責任
